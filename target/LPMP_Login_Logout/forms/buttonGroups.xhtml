Button Groups and Toolbars

BootsFaces lets you group a series of buttons together on a single line using the <b:buttonGroup> component.

Basic example

Wrap a series of <b:button>s or <b:commandButton>s by enclosing them in a <b:buttonGroup> .

JSF markup:

Button toolbar

You can go further and obtain a Toolbar combining sets of <b:buttonGroup>s inside a <b:buttonToolbar>

JSF markup:

Sizing

Instead of applying button size attribute to every button in a group, just add the size attribute to the <b:buttonGroup>.

JSF markup:

Mixed Button Groups

If you want dropdown menus mixed with a series of buttons, just add the <b:dropButton> inside the <b:buttonGroup>.

You can even use different looks and/or icons for your buttons, if you need to.

JSF markup:

Vertical variation

Make a set of buttons appear vertically stacked rather than horizontally, using the orientation="vertical" attribute.
